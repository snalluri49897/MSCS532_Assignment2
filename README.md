# Assignment 2
### Steps to Run the Code
1. **Download the file** using the download button or clone the git repository.
2. **Make sure Python is installed** — open a terminal and check:
   ```
   python --version
   ```
   If not installed, grab it from [python.org](https://python.org). You need 3.6+.

3. **Run it:**
   ```
   python divide_and_conquer.py
   ```
   No external libraries needed — everything used (`time`, `random`, `tracemalloc`, `sys`, `math`) is part of Python's standard library.

That's it. The script will print the correctness checks, both benchmark tables, and the growth rate analysis straight to your terminal.
